# Caroline Somers

Cybersecurity-interested professional currently working in network security. In my free time I do reverse engineering, malware analysis, and CTFs.

I enjoy working through CTF challenges as a way to study how cryptographic and memory-safety assumptions break in practice. Full writeups (vulnerability analysis, exploitation, remediation) are in [carolinesomers/writeups](https://github.com/carolinesomers/writeups).

**A few I'd point to first:**
- [`rsa_oracle`](https://github.com/carolinesomers/writeups/tree/main/picoctf/crypto/rsa_oracle) — chosen-ciphertext attack recovering an AES key from a textbook-RSA oracle
- [`bit-burger`](https://github.com/carolinesomers/writeups/tree/main/ritsecctf-2025/pwn/bit-burger) — turning an uninitialized function pointer into an arbitrary-call primitive to bypass an auth check
